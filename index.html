Important font stuff

For an accurate font, you'll need to have the Bodoni Highlight ICG font installed. Otherwise, a font from Google will be used, which requires internet access. Either way, no fonts are embedded, so if you want to guarantee that a special font will be used you should export as PNG (high-resolution would be best of course).
